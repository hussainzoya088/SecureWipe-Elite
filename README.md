🚀 SecureWipe Elite
Enterprise Data Sanitization & Real-Time Monitoring Dashboard
📌 Overview

SecureWipe Elite is a cybersecurity project that provides secure data wiping and real-time system monitoring through a visually rich dashboard interface.

The system focuses on:

Preventing data remanence
Providing secure deletion of files
Monitoring system activity in real time
Simulating enterprise-level SOC (Security Operations Center)

This project is designed for academic submission, cybersecurity labs, and demonstration purposes.

🎯 Features
🔴 Secure Data Wiping
Overwrites files using random data
Prevents recovery of deleted data
“Execute Destruction” feature from dashboard
📊 Live Dashboard (HTML-Based)
